About the Project

What is UofT Tools?

UofT Tools is an unofficial, open-source suite of utilities built by a student to make everyday campus life at the University of Toronto a little easier. The first tool is an automatically-updated academic calendar feed for all three UofT campuses (UTSG, UTM, UTSC). More tools are planned.

Is this an official University of Toronto product?

No. This project is not affiliated with, endorsed by, or supported by the University of Toronto in any way. It is an independent student side-project. Always verify important dates and deadlines on your campus’s official registrar website.

Who built this and why?

Built by a UofT student. The original motivation was accessibility: important academic dates are buried across multiple registrar pages with no machine-readable export, making them hard to track — especially for students who rely on calendar reminders for time management and planning. The source code is fully open on GitHub.

Is this an accessibility tool?

Yes, by design. Delivering academic dates as a standard calendar feed means students can receive system notifications, use screen readers, integrate with assistive scheduling software, and generally manage their time in whatever way works best for them — without having to manually visit and parse university web pages. Accessibility is a core design goal, not an afterthought.

Using the Tools

How do I subscribe to the academic calendar feeds?

Go to the Calendar Feeds page, pick your campus, and click Add to Calendar. The webcal link works with Google Calendar, Apple Calendar, Outlook, and most other apps. Your calendar will auto-update daily without any action on your part.

How often are the calendars updated?

Once every 24 hours via a scheduled GitHub Actions job. The scraper fetches the latest data from each campus registrar page, regenerates the ICS files, and commits them to the repository. Cloudflare serves the updated files immediately after.

What if a date in the calendar is wrong?

The dates are scraped directly from official registrar pages — if the registrar page is correct, the calendar should be too. If you spot a discrepancy, please open an issue on GitHub with a link to the official source. In the meantime, always trust the official page over this tool.

Is the project free to use?

Yes, entirely. There are no accounts, no tracking, and no fees. The project is open-source under the MIT licence.

Contributing & Feedback

How do I report a bug or request a feature?

Open an issue on the GitHub Issues page. Please include as much detail as possible — what you expected to happen, what actually happened, and a link to any relevant official source if the issue is about incorrect dates.

Can I contribute code?

Absolutely. Pull requests are welcome. The repository is at github.com/wuyilingwei/UofT-Tools. Whether it’s fixing a scraper, improving the UI, or adding a brand-new tool — contributions of all sizes are appreciated.
