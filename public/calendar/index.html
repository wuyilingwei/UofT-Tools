ⓘ How it works

  • A GitHub Actions job runs once per day.
  • UTM — headless browser renders the Elfsight widget on the UTM registrar page.
  • UTSC — static HTML tables on the UTSC registrar sub-pages are fetched directly.
  • ArtsCI — headless browser expands the accordion panels on the ArtsCI dates page.
  • All events are converted to all-day ICS entries and committed to this repository.
  • Calendar apps that support webcal/CalDAV poll the URL and pull in any changes.
